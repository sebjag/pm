沙

忠心的约伯描述自己烦恼不堪，说：“我的愁烦比海沙还重”。（伯6:3）平均来说，只是1立方米（1.3立方码）的湿沙就重1900公斤（4200磅）了。虽然沙土沉重，但愚蠢人为别人带来的愁烦比沙土还要沉重难担，箴言27:3就曾提到这点，说：“石头沉重，沙土难担，愚蠢人造成的愁烦比这两样更沉重难担。”

药蜀葵

一种多年生植物，跟蜀葵同科。药蜀葵（学名Althaea officinalis）的木茎可长到1.8米（6英尺）高。叶宽大，边缘为锯齿状，前端钝尖。茎和叶都披上一层柔软的绒毛。花淡粉色，开5瓣，直径约5厘米（2英寸）。根白色，形状像胡萝卜，在饥荒时曾用作食物。在圣经里，药蜀葵只出现过一次，经文暗示它淡而无味。（伯6:6）

“哈拉穆特”只出现于约伯记6:6，有些译本翻做“蛋”（和合，新译，现译），有些翻做“马齿苋”（吕译），L．凯勒和W．鲍姆加特纳合编的希伯来语和阿拉米语词典则界定为“药蜀葵”。（《旧约全书辞典》，莱顿，1958,304页）

劳役

希伯来语tsa·vaʼʹ（察巴）常指在军队或战争中服役，也可以指服劳役抵债或抵罪。例如，先知告诉耶路撒冷，这城的“服役期”满了，罪过赎清了。（赛40:1,2，Rbi8 脚注）受考验期间，痛苦不堪的约伯把生命比作辛苦而让人疲累的“劳役”。约伯问：“地上的凡人，不是要服劳役吗？人的日子，不是像雇工的日子吗？”（伯7:1）带着类似的感受，后来他对上帝说：“你会……越发恼怒我；磨难接二连三地临到我身上［或译：我得一次又一次地服劳役］。”（伯10:17，Rbi8 脚注）看来约伯认为，上帝使磨难一次又一次地临到他，让他苦上加苦。约伯也把死人留在坟墓里的时间比作劳役，是被迫承受的重担，不过他对复活满怀希望。（伯14:14）

约伯记经文选讲

7:1；14:14 “服劳役”是什么意思？“服劳役的日子”又意味着什么？ 约伯饱受打击，痛苦不堪，以致认为生活就像艰苦的劳役一样。事实上，约伯记10:17中谈到的“磨难”也可译做“劳役”。另外，人从死后到复活都得在死亡中“沉睡”，而“沉睡”时间有多长不是死去的人可以自行决定的，就像人不能自行决定服劳役的日子有多长一样，所以约伯把人在死亡中“沉睡”的日子比作服劳役的日子。

你难于入睡吗？

失眠的方式大致分为三类。有些人在上床时起初极难入睡。有人很快睡着，但很早便醒来而无法再入梦乡。有些人的难题是夜里醒来多次。

失眠的原因

失眠的原因是什么呢？数百种原因中的任何一种都可以造成失眠。它可能是遗传的，即脑中先天地缺乏一种称为，“斯路吐连”的荷尔蒙状分泌物。这种分泌物的作用据说很像“催眠液”，它进入血里使身体困倦思睡。严重的痛苦，如圣经中族长约伯所患的，也能中和“催眠液”，因为他曾说，“我尽是反来覆去，直到天亮。”——约伯记7:4。

你也许没有此种遗传缺陷，但你的难题或者是由轻微的痛苦造成，这些痛苦是在忙碌的日间所忽略的，可是在夜间却驱之不去，以致使你醒来多次。你的睡眠难题也许由于房间不够通风，床褥太硬或太软。

在另一方面，你难以入睡可能是由于刺激品如咖啡、茶或可乐类饮品所致。或者，你在上床之前有吃得太饱或吃难于消化的食物的习惯。也许刚相反，饥饿使你睡不着，甚至便秘有时也会妨碍睡眠。

消极的情绪如罪咎感、过度的野心、情绪不安，尤其是焦虑和担心，都可使人不眠。正如圣经说:“富足人的丰满，却不容他睡觉。”——传道书5:7,12。

精神颓丧也能导致失眠，尤以神经疲劳为甚。仇恨或竞争意识亦使人失眠，虽然是出于不知不觉的。在另一方面，过度的娱乐和兴奋或追寻剌激也可以夺去你的睡眠。

安眠药又如何？

服食安眠药看来是最简易的解决办法，但是要小心！它们只能在紧急时候运用。在一两星期内，它们也许有所帮助，但大多数作用还是害多于利。因此斐利文医生在《睡眠研究》一书中说:“胡乱作安眠药处方是医药界最普遍的错误。”

那末人们为什么要继续服用安眠药呢？也许因为他们将之当作定心丸;受害者想象自己受到安眠药的帮助。更有可能的是，吞丸一经成为习惯，则“戒断症状”可能颇为难受以致成为服用理由。是故医生劝告人戒断安眠药要采取逐渐戒除方式。

克服失眠症

失眠症是不必倚靠药丸也可以克服的。方法先由较基本的事情开始，要肯定你的寝室有适当的空气流通，床褥既不太软也不太硬。噪音会使你睡不着，或睡着之后会将你吵醒吗？这样，耳塞也许对你有所帮助。一位研究睡眠权威人士推荐让电风扇整夜开着，它那有规则的声音会发生安抚作用，并且消除不规则的噪音。

控制饮食对于好睡也颇重要。也许你需要在午后避免饮咖啡或可乐之类的饮品，必要时可戒绝这一切。同时，你若有“甜食嗜好”便要少些摄取甜品。以大多数人而论，在黄昏时候不吃太丰富的餐或太辛浓的食物是值得推荐的。在这时候可吃容易消化的食物。

不少人发现在睡前进点饮品有助于酣睡，例如热奶、麦精饮品、混和糖蜜或蜂蜜的薄荷茶，或以安眠效果著称的草药饮品等。有些人发觉热葡萄汁，也有人认为一杯啤酒或葡萄酒有助于安眠。更有些人求助于手疗或整骨疗法。已婚夫妇可以借着彼此以按摩方法解决入睡难题。

适当的运动对克服失眠也有帮助。你若从事久坐职业，“要逐渐增加日间运动的水平”，基利医生在1974年九月四日的《医药论坛》如此说。对于这项劝告，圣经指出:“劳碌的人……睡得香甜。”（传道书5:12）可是，睡前却要避免剧烈运动，因为它除了使你疲倦之外，也能使你兴奋。将近就寝时的最佳运动是步行或伸展四肢。

编织

织机上的经线通常会相间归集成两组，提升其中一组，就可在上下两组之间形成“梭道”。纬线在梭子（牵引纬线的工具棒）牵引下从上面那组经线表面穿过，再从同一组经线的下面穿回来。这样就需要两条“梭道”。简易的水平织机用一条扁平的“分纱杆”将所有经线一条上一条下地相间分为两组。转动“分纱杆”就会产生一条“梭道”，让纬线可以穿过。而下面的那组经线被一个个线圈（“线综”）一条条地连到一根“提综棒”上，“提综棒”放在经线的上面，提起“提综棒”就会产生另一条“梭道”，让纬线可以从经线中再穿回来。纬线每次穿过经线后，都要用打纬刀把纬线向织好的那边压紧。编织工用梭子把纬线穿过经线。熟练的编织工运梭如飞，所以约伯说：“我的日子比梭还快”。（伯7:6）

风

用做比喻 风起得快，走得也快，用来比喻人生苦短十分恰当。（伯7:7）风没有实质，因此也指空洞的知识、徒劳的工作、无法兑现的承诺和希望（伯15:1,2；16:3；传5:16；何12:1），以及没有价值的事物（赛26:18；41:29；耶5:13）。无益的事到头来尽是虚空，为这些事奔波的人犹如“捕风”。（传1:14；2:11）“连累家人受排斥的，产业如风”，意思是无论这个人得到什么，都是没有任何真正的价值的。（箴11:29）

约伯记经文选讲

7:9,10；10:21；16:22 约伯这些话表示他不相信复活吗？ 约伯这几句话是就自己不久的未来而言。他的话是什么意思呢？意思也许是，如果他死了，跟他同年代的人就不会再看见他。从他们的角度来看，在上帝指定的时候来到之前，约伯再也不会回家，别人也不再认识他了。约伯的意思也可能是，人死后绝不能够凭自己的力量重得生命。事实上，约伯记14:13-15清楚表明，约伯相信死了的人有希望在未来复活。

[ 1989/9/15刊5页 - this link has citations, only first citation shown, click link to see more]

“我们知道在复活的时候，他们必会活过来”

但一些熟悉圣经的人也许反驳道：“嗯，等等，约伯不是在约伯记16章22节说他‘必走那往而不返之路’吗？在约伯记7:9他也指出：‘人下阴间也不再上来。’约伯也在第10节补充道：‘他不再回自己的家；故土也不再认识他。’”

故此，正如有些学者声称，这些经文和类似的话不正显示约伯认为死亡是个“往而不返之地”吗？究竟这些话是否表示约伯并不相信未来有复活呢？为了解答这个问题，我们看事情时必须把有关的背景一同考虑，并且要和约伯就这个问题所表达的其他思想互相比较。

约伯并不知道他受苦背后的原因。他曾经误以为上帝要对他的磨难负责。（约伯记6:4；7:17-20；16:11-13）由于意志消沉，约伯觉得他惟一可以立刻得到舒解的地方就是坟墓。（约伯记7:21；17:1；参阅3:11-13。）所以，从和约伯同时代的人的观点看来，他们不会再见到约伯，约伯也不会再回家，不再为人认识，不再重返人世，在上帝所指定的时间来到之前约伯毫无希望。假如上帝对此事不插手干预，任由人自生自灭，则无论约伯抑或其他任何人，都没有能力从死里复活。a——约伯记7:9,10；10:21；14:12。

约伯并不知道他受苦背后的原因。他曾经误以为上帝要对他的磨难负责。（约伯记6:4；7:17-20；16:11-13）由于意志消沉，约伯觉得他惟一可以立刻得到舒解的地方就是坟墓。（约伯记7:21；17:1；参阅3:11-13。）所以，从和约伯同时代的人的观点看来，他们不会再见到约伯，约伯也不会再回家，不再为人认识，不再重返人世，在上帝所指定的时间来到之前约伯毫无希望。假如上帝对此事不插手干预，任由人自生自灭，则无论约伯抑或其他任何人，都没有能力从死里复活。a——约伯记7:9,10；10:21；14:12。
