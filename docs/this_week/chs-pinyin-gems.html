结盟，结亲

犹大王亚撒用王宫宝库的金银收买叙利亚王便哈达一世，要他废除跟以色列王巴沙所立的约。（王上15:18-20）先知哈拿尼责备亚撒宁愿“依靠叙利亚王”也不依靠耶和华，说：“你既然做了这件愚蠢的事，从今以后，你必常遭战祸。”（代下16:7-9）后来，以色列王亚哈打败便哈达二世，跟他立约，同样被上帝的先知谴责。（王上20:34,42）约沙法跟亚哈结盟，联手攻打叙利亚，并没有得胜。先知耶户质问约沙法说：“你怎可以帮助恶人，爱那些恨恶耶和华的人呢？耶和华为此向你降怒了。”（代下18:2,3；19:2）另一次，约沙法跟作恶多端的以色列王亚哈谢结盟，合伙造船。但后来船只毁坏，上帝通过他的先知所作的预言果然应验了。（代下20:35-37）犹大王亚玛谢听从上帝的劝告，作出英明的决定，解散从以色列雇来的部队，只是也因此损失了3400公斤银子（66万零600美元）。（代下25:6-10）

以法莲

以法莲既是北方的以色列国中最有势力的部族，常被用来代表北方的整个十部族以色列国（代下25:7；耶7:15），所以先知何西阿和先知以赛亚都曾强烈谴责以法莲。何西阿指责以法莲跟外邦各族混杂，仿效他们的行为，事奉他们的偶像。他把以法莲比做没有翻过的圆饼，下面已经烤熟甚至烤焦，上面却还没有烤过。（何7:8；另见诗106:35,36；何4:17；12:14）虽然被外族人吞吃了力量，以法莲却没有回头归向耶和华，反而向埃及求助，跟亚述结盟。这样，以法莲就像一只愚蠢无知的鸽子，落在网罗里，无法逃生。（何7:9-12；8:9；另见王下17:4；何12:1）

罪行和刑罚

亚哈谢

历代志下22:7说，亚哈谢王遇害，“是出于上帝”。亚哈家已被上帝定了罪，亚哈谢王却跟这家同流合污，耶户剪除他，其实是执行上帝的判决。在历代志下22:6，亚哈谢称为“亚撒利雅”（不过，有十五部希伯来语抄本仍沿用“亚哈谢”）；在历代志下21:17和25:23，亚哈谢称为“约哈斯”（即其中所含的上帝名字从后缀被改为前缀）。

[ 1066-1067页 ]

门

以法莲门 以法莲门位于宽墙，在角门东面400肘尺（178米；583英尺）。（王下14:13；代下25:23）这是城北的出口，朝着以法莲部族的领土。同样，有些学者主张以法莲门是中门（耶39:3），另一些则主张是第一门（亚14:10）。一般认为，以法莲门是（或类似于）犹太历史家约瑟夫斯所提及的“热纳”或花园门。（《犹太战记》第5卷146节［4章2段］）以法莲门附近有一个广场，在尼希米的日子，人们在那里搭棚庆祝住棚节。（尼8:16）在重建城墙的记载中，尼希米没有提到以法莲门，显然是因为这道门不需要大规模修缮。

角门 这道门显然位于城墙的西北角，以法莲门的西边。（王下14:13；代下25:23）角门在欣嫩谷的东侧，看来位于旧城西墙跟宽墙的连接处。乌西雅在这道门附近建造了一座塔楼，圣经没有指明是不是炉楼。（代下26:9）从耶利米和撒迦利亚的话看来，角门位于城的西面边界。（耶31:38；亚14:10）

你身边有属灵的良师益友吗？

乌西雅年纪轻轻，16岁就登上国位，统治南部的犹大国。他从公元前9世纪后期开始统治直到公元前8世纪早期，在位超过50年。乌西雅年少时“行耶和华看为对的事”。谁对他产生积极的影响，使他行走正直的道路呢？圣经记载说：“撒迦利亚在世的日子，教导乌西雅敬畏上帝，乌西雅就一直寻求上帝。在他寻求耶和华的那段日子，上帝叫他富强起来。”（历代志下26:1,4,5）

除了上述的圣经记载，我们对撒迦利亚所知并不多，只知道他是劝导乌西雅的良师。由于他“教导乌西雅敬畏上帝”，年轻的国王受到良好的影响，行事正义。《注释者圣经》（英语）说，撒迦利亚显然“通晓神圣的话语，有丰富的属灵经验，也善于传授知识”。一个圣经学者对撒迦利亚作了以下结论：“他熟谙预言……深具智慧，为人虔诚，处事正直。看来撒迦利亚对乌西雅产生了极大的影响。”

乌西雅忠于上帝，结果得享厚福。由于“上帝帮助他”，他“能力非凡”。是的，“撒迦利亚在世的日子”，乌西雅忠心事奉上帝，他因此在许多方面取得很大的成就。（历代志下26:6-8）可是他一强盛，就把良师撒迦利亚的教导抛诸脑后。乌西雅“心高气傲，自招败亡。他对耶和华他的上帝不忠”。有一次，他行事大大不敬，结果患上可怕的皮肤病，再无法全力执行君王的职务。（历代志下26:16-21）

在“寻求上帝”方面，你也有一位对你产生积极影响的良师吗？不论你是年轻还是已经长大，是弟兄还是姊妹，也可以有这样的一位良师。要好好珍视这位良师，因为他能勉励你继续做耶和华看为对的事。要听从这位成熟基督徒的劝导，认真考虑他的建议。他指导你“敬畏上帝”，你千万不要漠视他的智慧之言。（箴言1:5；12:15；19:20）

非利士

相关的预言 约珥的预言表明，由于非利士人把“犹大人”和“耶路撒冷人”卖给“希腊人”，非利士人也会有相同的下场。（珥3:4-8）既然约珥的预言看来在公元前9世纪写下来，非利士人败于乌西雅（代下26:6-8）和希西家（王下18:8）之手就可说是局部应验了这个预言。

不过，更大的应验显然在以色列人从巴比伦获释回乡之后。评注家C．F．凯尔说：“亚历山大大帝和他的继位人释放国内很多犹太战俘（参看德米特里王对约拿单的保证：‘我会释放那些被囚禁、被迫在我们国内做奴隶的犹太人。’约瑟夫斯《犹太古史》第13卷2,3章），非利士人部分地区和腓尼基人的土地都一度受犹太人管辖。”（《旧约评注》，1973,第10卷，约珥书，224页；参看俄19,20）值得注意的是，亚历山大大帝夺得非利士人的加沙城，许多居民被杀，其余的被卖为奴。不少别的预言也指出，耶和华会向非利士人施行报应。（赛14:31；耶25:9,20；47:1-7；结25:15,16；摩1:6-8；番2:5；亚9:5-7；详见亚实突；亚实基伦；以革伦；迦特；加沙1）

门

角门 这道门显然位于城墙的西北角，以法莲门的西边。（王下14:13；代下25:23）角门在欣嫩谷的东侧，看来位于旧城西墙跟宽墙的连接处。乌西雅在这道门附近建造了一座塔楼，圣经没有指明是不是炉楼。（代下26:9）从耶利米和撒迦利亚的话看来，角门位于城的西面边界。（耶31:38；亚14:10）

从角门到西南墙的谷门之间是西墙，圣经没有提到西墙上有城门。这无疑是因为欣嫩谷的斜坡十分陡峭，所以在西墙建造城门不切实际。尼希米记没有提到角门，也许同样是因为这道门不需要大规模修缮。但记载提到修复炉楼，炉楼看来是角门的一部分，或在角门附近。（尼3:11）

坚固城

除了在城中，一些偏僻的地方也建造塔楼（希伯来语migh·dalʹ米格达；复数，migh·da·limʹ米格达林）。这些塔楼有如“警察哨所”，守卫水井或其他水源、道路、边境、运输线或补给线。犹大王乌西雅在耶路撒冷和旷野建造塔楼，看来是为了守卫附近的水池，那些水池是乌西雅凿来贮水给牲畜喝的。（代下26:9,10）在南地也发现了一些这样的塔楼。

耶路撒冷

乌西雅王（公元前829-前778）为耶路撒冷设置了更多坚实的防御工事。他在（西北）角门和谷门（在西南角）建造塔楼，加强城门的防御能力。又在“扶壁”（“拐角处”，吕译；“转弯之处”，和合）建造塔楼，“扶壁”看来是东面某段城墙，离大卫或所罗门的王宫不远。（代下26:9；尼3:24,25）乌西雅还在塔楼和墙角放置“军械”，可能是用来发射利箭和投掷大石的弹射器。（代下26:14,15）他儿子约坦继续进行各项建造工程。（代下27:3,4）

高傲

一个人即使向来谦卑为上帝服务，也可能会变得自高自大，原因也许是财富多了，权力大了，又或者自觉相貌、成就和才智比人优越，还有就是受到别人赞许。犹大王乌西雅就是一个例子。乌西雅本来勤政爱民，一直蒙耶和华赐福。（代下26:3-5）圣经记载：“可是，他一强盛，就心高气傲，自招败亡。他对耶和华他的上帝不忠，竟闯进耶和华的殿，要在香坛上烧香。”（代下26:16）乌西雅心高气傲，擅自在坛上烧香，做了惟独祭司才有权做的事；按照上帝的规定，君王和祭司各有各的职责，界线分明。

    埋葬，坟地

    根据列王纪上2:10，大卫是葬在“大卫城”的，看来这个地方更成了犹大列王共用的坟地。大卫以后相继有20个王，圣经直接提到其中12个被葬在大卫城，不过这不表示他们都葬在“列王的坟地”，例如约兰，约阿斯（又名约阿施）以及亚哈斯就明显不是了。（代下21:16,20；24:24,25；28:27）其实，“列王的坟地”很可能是指大卫城内的某个地区，即列王的坟墓的集中地，不是指某个分成不同墓室的大陵墓。圣经记载，亚撒王“葬在大卫城他为自己所凿的陵墓里”（代下16:14），希西家则“葬在大卫子孙的山陵上”。（代下32:33）患麻风病的乌西雅王“跟列祖葬在同一个地方，但只埋在列王坟场的地里，因为人们说：‘他是个麻风病人’”。这段记载看来显示，乌西雅的遗体被土葬，而不是放在凿出来的岩洞里。（代下26:23）

    [ 《洞悉圣经》（下册）453页 ]

    乌西雅

    论到乌西雅死后下葬，历代志下26:23说：“最后乌西雅随列祖长眠，跟列祖葬在同一个地方，但只埋在列王坟场的地里，因为人们说：‘他是个麻风病人。’”这话的意思可能是，由于乌西雅长了麻风，他被葬在王家坟场的一块地里，而非葬在从岩石凿出来的墓穴中。

    在耶路撒冷出土的一块石灰石碑铭（据称成于公元1世纪），上面刻着以下一句话：“这里是犹大王乌西雅的骸骨所在。不可开启。”（见上册960页图片）

    门

    “耶和华殿的上门” 这也许是通往内院的一道门，是耶和华殿的“新门”。耶利米在这道门前被审讯，耶利米的书记巴录也在这里向民众宣读书卷上的话。（耶26:10；36:10）耶利米把这道门叫做“新门”，也许是因为这道门不像其他门一样古老。这道门可能是约坦王建造的“耶和华殿的上门”。（王下15:32,35；代下27:3）

    大麦

    大麦比较便宜，是穷人主要的粮食。不过，在以色列，人们即使买得起小麦，也不会把大麦视为次等的食物。大卫王和他的随从躲避押沙龙的追杀，逃到基列。当地人送来日用品和食物，其中包括大麦。（撒下17:27-29）泰尔王的仆人为所罗门准备建造圣殿的材料，所罗门就给他们大麦2万歌珥（440万升；12万5000蒲式耳），小麦2万歌珥，还有大量的油和酒，并派人把这些物品送到希兰那里。（代下2:10,15）亚扪人的王向犹大王约坦进贡，贡物包括1万歌珥（220万升；6万2500蒲式耳）大麦。（代下27:5）耶路撒冷陷落后，有十个人求刺客以实玛利不要杀他们，说他们“在田野里有个宝藏，里面藏了小麦、大麦，油和蜜”。（耶41:8）

    亚扪人

    以色列国分裂后 约沙法王在位期间（公元前936-约前911），亚扪人摆脱了大卫王朝的君王的辖制，跟摩押人和西珥山人一同攻打犹大，结果惨败而回。（代下20:1-4,10-26）据亚述王撒缦以色三世（跟公元前904年-前877年期间作以色列王的耶户同时代）的铭文透露，在甲加战役中一起对抗亚述的联军包括了“亚扪人鲁胡比之子巴萨”率领的大军。（《古代近东文献》，J．B．普里查德编，1974,279页）另外，合谋杀死犹大王约阿施（约公元前859）的人中也有亚扪人，即亚扪妇人示米押的儿子撒拔。（代下24:22,26）乌西雅在位期间（公元前829-前778），犹大国国富民强，亚扪人再次被迫向犹大王进贡。（代下26:8）乌西雅的儿子约坦辖制亚扪人，要他们献上3400公斤银子（66万零600美元）、220万升小麦和220万升大麦。（代下27:5）亚扪人能连续三年献上大量的贡品，跟他们所占的地利不无关系。亚扪人的土地位处阿拉伯与大马士革商贸往来的要冲，雅博溪谷一带也比较肥沃，而且大麦小麦就是当地的主要农产品。

    读者来函

    圣经也提及另一些书，虽然书名跟某些经卷的名称很相似，但没有收录在圣经中。为了说明这点，请留意以下四卷古籍：《犹大王大事记》《犹大王与以色列王记》《以色列王记》《以色列王与犹大王记》。这几卷书的名称看起来跟我们所熟悉的列王纪上和列王纪下很相似，但全都不是受圣灵启示的，也不是圣经正典的一部分。（列王纪上14:29；历代志下16:11；20:34；27:7）这些古籍只是当日的历史著作，耶利米先知和以斯拉执笔写圣经时，可能参考过这些古籍。
